# ID‑Generator – Respo‑Rotation Name Engine

Der ID‑Generator erzeugt neue Projekt‑Namen im Stil des Respo‑Rotation‑Systems.  
Er nutzt eine **Markov‑ähnliche Silbenmischung**, kombiniert mit der **DNA bestehender Projekte**:

- Respo‑Rotation  
- Scan‑Ghost  
- BOERSE  
- TP‑System (TP4 / TP8 / TP12)  
- ECO‑EXP  
- Scan81  
- ki3KIme  

Damit entstehen **echte neue Respo‑Projekte**, die plausibel und systemkompatibel sind.

---

## 🔹 Funktionsweise

Der Generator mischt:

1. Basis‑Projekt‑Namen  
2. Respo‑Silben (Markov‑ähnlich)  
3. Rotation‑Silben (Spin, Flow, Loop, Whirl)  
4. Whirlpool‑Power‑Silben (Boost, Whirl)  
5. Zufalls‑Suffix (Projekt‑ID)

Beispiel:

WhirlBoost-Respo-9FQ2
SpinFlow-TP8-K3L9
GhostLoop-BOER-7XQ1

## 🔹 Zweck

Der ID‑Generator dient dazu:

- neue Respo‑Rotation‑Module zu benennen  
- neue BOERSE‑Tools zu erzeugen  
- neue TP‑System‑Addons zu generieren  
- neue Ghost‑Scan‑Projekte zu erstellen  

Er ist ein **Name‑Engine** für dein gesamtes System.
